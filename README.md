## Custom Keybinds

### <C->

- C-space   => Incremental selection Treesitter (use <bs> to decrement) 
- C-p       => Files Files
- C-_ (/)   => Comment Line(s) 

### <leader>p(roject)

- pe        => Project Explorer
- pf        => Project Files
- ps        => Project Grep search
